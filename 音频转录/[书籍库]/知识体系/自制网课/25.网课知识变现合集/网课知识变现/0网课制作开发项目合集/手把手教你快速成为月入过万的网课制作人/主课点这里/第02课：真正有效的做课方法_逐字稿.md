# 第02课：真正有效的做课方法.mp3 逐字稿

> 时长: 07:46 | 转录时间: 2026-06-27 04:40

**`[00:02]`** Hello 大家好,欢迎继续收听莫提网课制作系列课堂的第二课,真正有效的作课方法及步骤。

**`[00:11]`** 我呢是一个善于学习和总结方法论的人,又买了很多作课相关的书籍研读,还买了很多网课来学习。

**`[00:26]`** 一边搜集资料,一边时间作课,慢慢的琢磨出了一套行之有效的方法。

**`[00:32]`** 首先第一个作课小方法,搜私并用法。

**`[00:37]`** 什么是搜私并用法呢?顾名思义就是一边搜索,一边思考。

**`[00:44]`** 作课离不开搜索资料素材,也离不开思考,边搜索,边思考,可以在由被动转化为主动的过程中逐步加深对课程内容的认知和规划。

**`[00:57]`** 那么我平时是如何搜索资料的呢?一般我会去用B应国内版的搜索引擎,国外版的搜索引擎用谷歌。

**`[01:06]`** 用这两个搜索引擎跳过了百度的广告,比较快速的搜索到有用的答案。

**`[01:12]`** 我经常用来找资料的网站名字叫做重部落快搜网站。

**`[01:17]`** 重部落集合了Google、百度、B应等国内外综合网站,是找学术、资源、专业领域知识或垂直干货的好网站。

**`[01:27]`** 在这里大家千万不要小看搜索,精通搜索能解决80%的问题。我认为人人都需要学会搜索,搜索是一项必备技能。

**`[01:39]`** 但是只搜索不思考就等于是只搬运不吸收,谈不上真正转化为自己的知识。

**`[01:47]`** 所以搜索的过程中要思考,因此我给这个小方法命名为搜私并用法。

**`[01:55]`** 第二个作客小方法,快速看书法。快速看书对作客有什么用呢?

**`[02:01]`** 快速看书实际上可以帮助你寻找到作客的灵感。

**`[02:06]`** 本质上知识付费就是顺应时代潮流,把书籍的内容做成了音频和视频来进行售卖。

**`[02:14]`** 所以我们要开始做一门课的时候,快速看书是很有必要的。快速看书看的是什么呢?

**`[02:22]`** 看的是书的标题,每个章节的设置以及小标题的设置,大概粗略的浏览书本内容,从中寻找到作客的灵感。

**`[02:32]`** 一个人比较常用的是网易云阅读这个app,我也经常在上面看书寻找作客灵感。

**`[02:40]`** 众筹新客联系微信,501863613。

**`[02:49]`** 第三个作客小方法,听写并用法。多去听同行老师的课,听的过程中写下别人课程的优缺点并进行修改。

**`[02:59]`** 如果是你来做,你会如何做,并且写下改进意见。这样一来你就能够慢慢的形成自己的作客逻辑。

**`[03:08]`** 同时去看别的网课章节设计小标题设计,对比来看哪个章节设计更好,哪个小标题设计得更有意思。

**`[03:18]`** 把你认为有意思的地方写下来,等到自己作课的时候就可以借鉴啦。

**`[03:23]`** 这种快速收到效果的方法拿来就可以用。推荐腾讯课堂这个网站,上面有许多不同种类的课程可以听,可以帮助初步了解课程的内幕。

**`[03:35]`** 以上三点是我常用的并且有效的快速作客的方法,分别是搜集作客素材,寻找作客灵感,整理作客逻辑三个方面。

**`[03:47]`** 讲完作客的小方法,我们接着往下讲,讲一讲五个具体的作客步骤。

**`[03:55]`** 在我们作客的过程中,大约会经历五个步骤。

**`[03:59]`** 第一个步骤,选题。

**`[04:02]`** 我们来回应一下,上学的时候写的最多的作文,一般作文分为两种,一种是固定标题作文,一种是自你标题作文。

**`[04:13]`** 作课也是一样的道理,要先有标题,标题又概括了整个课程的意思。

**`[04:20]`** 不同的是,上学的时候,老师给的作文标题还有固定的。

**`[04:25]`** 现在自己来作课,标题不再有人给你订好,你得自己来选标题。

**`[04:31]`** 你想做什么题目的课程是你的自由,任由你发挥。

**`[04:36]`** 另外,一门课程又好比一本书,所以我们做的网课也像一本书一样。

**`[04:42]`** 一本书偏向表达的内容是什么?是爱情的,是立志的,是考试的,还是工具书?

**`[04:49]`** 同样,一门课程也有偏向表达的内容,这里面就涉及到选题。

**`[04:56]`** 我们再把它分解一下,选择你要讲的课程主题标题就叫做选题。

**`[05:02]`** 选题也是至关重要的一步,后面第九课还会详细来讲选题。

**`[05:08]`** 第二个步骤,备课。

**`[05:12]`** 确定好了课程的标题,我们下一步就要进行备课了。

**`[05:18]`** 顾名思义,备课就是准备课程的内容。

**`[05:21]`** 一堂有料的课,内容肯定是要经过事先好多天,甚至是几个月的精心准备。

**`[05:28]`** 内容包含了图片、文字的构成,我们把这个部分称之为准备初步稿件。

**`[05:35]`** 第三个步骤,制作PPT换灯片。

**`[05:39]`** 有了出稿之后,我们就要开始制作正式对外展示的PPT。

**`[05:44]`** 第四个步骤,预讲。

**`[05:48]`** 为什么要做预讲呢?这个实在是太有必要了。

**`[05:52]`** 预先多讲几遍,自己或朋友、家人感受一下你的语气、神态、节奏感。

**`[05:58]`** 都有什么地方需要改进的,哪些是可以保留的。

**`[06:02]`** 这样子等真正开讲的时候,自然也就更有底气。

**`[06:07]`** 第五个步骤,正式讲。

**`[06:12]`** 辛辛苦苦做出来的课件终于要跟学员见面了。

**`[06:18]`** 进行到正式讲课录制的阶段,录制的课程又分为音频和视频两种。

**`[06:24]`** 对自己的声音和普通话不够自信,还可以找专业的播音员来进行录制课程。

**`[06:31]`** 以上就是详细的五个做课步骤。

**`[06:34]`** 拆解完了做课的步骤,下面我们接着讲做课需要用到的工具和软件。

**`[06:40]`** 在进行课程录制的时候需要用到电脑、耳麦、话筒、音频视频录制软件。

**`[06:47]`** 其中音频视频录制软件有Camtasia Studio或Snagit或Cyberlink Ucam、PPT软件。

**`[06:56]`** 不那么专业的录制也可以不用耳麦,用话筒就够了。

**`[07:00]`** 最后我们来总结一下真正有效的做课方法和做课步骤。

**`[07:05]`** 其中有三个小方法和五个小步骤。

**`[07:09]`** 三个小方法分别为搜私并用法、快速看书法、听写并用法。

**`[07:15]`** 五个小步骤分别是选题、备课、制作PPT幻灯片、预讲、正式录制。
