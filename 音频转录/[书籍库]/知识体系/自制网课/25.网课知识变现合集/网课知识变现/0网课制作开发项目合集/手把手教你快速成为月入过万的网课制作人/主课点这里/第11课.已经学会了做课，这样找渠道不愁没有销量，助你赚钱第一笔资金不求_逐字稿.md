# 第11课.已经学会了做课，这样找渠道不愁没有销量，助你赚钱第一笔资金不求.mp3 逐字稿

**`[00:00]`** 哈喽大家好,欢迎来到莫提网课制作系列课堂第十一课,已经学会了做课,这样做课不愁没有销量,祝你赚取第一笔资金,不求人。准确的说,做好了课,那么就到上架销售阶段了。

**`[00:21]`** 这个时候有流量的自己卖,没有流量的呢,就找渠道分销,才是不愁没有销量,所以咱们先更正一下标题,已经学会了做课,这样找渠道不愁没有销量。

**`[00:37]`** 这节课我们着重讲,找渠道,首先是内容不错的课件,我们可以进行包装,并且放在我们平台lendingmaps来买,其次你也可以自己制作完成,再去找各种大大小小的平台分销。

**`[00:54]`** 那我们如何去找平台帮我们分销课程,下面我给大家推荐几个主要的大渠道,首先第一个,找今日头条,在今日头条开课,平台方有什么要求,你需要做什么。

**`[01:11]`** 首先,你需要在今日头条有一个头条号,这个头条号要有什么功能,头条号需要有图文远创标签,神明认证或者兴趣认证,满足这两个要求就可以开通专栏了,并且持续更新内容,在30天内更新大于10篇,且没有违规记录,就甄请图文远创标签。

**`[01:39]`** 开通了专栏之后,就可以上架你制作的付费课程啦。

**`[01:44]`** 今日头条的付费专栏里的章节,如果全部设置为付费,只能推荐给作者的粉丝,及关注作者的用户,可以在今日头条APP的信息流中刷到付费的内容。

**`[01:58]`** 所以作者拥有更多的粉丝,就意味着付费内容会被更多用户看到。

**`[02:04]`** 今日头条为了帮助作者获取更多的流量,推荐作者在专栏中设置部分免费章节,或设置部分章节为线面推广。

**`[02:17]`** 免费的章节和线面推广的内容,会推荐给头条里所有可能感兴趣的人,包括粉丝和非粉丝。

**`[02:26]`** 一个付费专栏下发布的内容超过5个,才会被推荐。

**`[02:31]`** 作者设置的线面推广的章节,会出现在今日头条APP的信息流中。

**`[02:39]`** 如果用户在信息流中点击该内容,可以免费阅读。

**`[02:44]`** 若用户对该专栏中的其他内容感兴趣,则需要付费阅读。

**`[02:50]`** 因此,线面推广是帮助作者为专栏引流的一项措施。

**`[02:55]`** 第二个,找网易云课堂。

**`[02:58]`** 如何成为网易云课堂的讲师呢?

**`[03:02]`** 在网易云课堂上面,可以提交个人讲师入住申请。

**`[03:07]`** 一到三天的审核期,审核通过就可以上传课件进行售卖了。

**`[03:13]`** 第三个,找腾讯课堂。

**`[03:16]`** 腾讯课堂上面有个人开课和机构开课。

**`[03:19]`** 个人入住腾讯课堂,只需要毕业证或者职业资格证等证件,其中一个就可以了。

**`[03:28]`** 第四个,找京东知识服务平台合作。

**`[03:32]`** 怎么入住京东知识服务平台呢?

**`[03:35]`** 京东本身是一个电商平台,所以它的模式是,在京东知识服务平台开知识店铺,还有门槛。

**`[03:44]`** 开店要交三万保证金,关店可以退还保证金。

**`[03:50]`** 平台管理费五百元一个月,也就是六千亿年,另外还要百分之二的销售额抽成。

**`[03:58]`** 第五个,找小额通。

**`[04:02]`** 小额通是用在微信端的知识店铺工具,收费比较贵。

**`[04:07]`** 基础的年费四千八亿年,还有其他各种功能也要收费。

**`[04:12]`** 第六个,立知微客,签聊live这两个是在微信端开知识店铺免费的工具。

**`[04:20]`** 先手网客这座人可以选择这个平台来开店,但是不好的地方是免费的东西也是最贵的东西。

**`[04:28]`** 如果你是一个公众号主,本身拥有不小的粉丝量,那么粉丝卖客之后,粉丝会被倒流到立知微客签聊他们平台。

**`[04:38]`** 第七个是知鸟,一个知识付费APP,做好的课程直接上传售卖。

**`[04:45]`** 下面讲一下入住知鸟平台的步骤。

**`[04:49]`** 第一步,打开知鸟网站。
