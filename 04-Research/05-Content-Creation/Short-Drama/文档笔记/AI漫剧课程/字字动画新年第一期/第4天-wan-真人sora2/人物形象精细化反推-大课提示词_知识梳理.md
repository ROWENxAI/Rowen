## 内容全文

## **一、核心理念**

```

<TEXT>

图片提示 = 零件清单 + 组装说明

```

当你需要AI生成**复杂精细的设计物**（手办、角色设计、产品、复杂道具）时：

> 不要描述感觉

不要留给AI想象空间

把每个零件写出来，像填BOM表

>

---

## **二、适用场景**

| **适用** | **不适用** |

| --- | --- |

| 手办/潮玩设计 | 风景照片 |

| 角色概念设计（需要完整设定） | 写意水墨 |

| 复杂道具/装备设计 | 氛围插画 |

| 产品结构图 | 情绪海报 |

| 需要"可制造"的设计稿 | 艺术表达 |

**判断标准**：如果这个东西要交给工厂生产，你需要告诉他们每个零件是什么——那就用清单穷举系统。

## **三、系统架构**

```

<TEXT>

用户创作意图

↓

【定性层】风格框架（这是什么类型的东西）

↓

【总述层】主体一句话概括（谁/什么/什么状态）

↓

【拆解层】分区零件清单（从上到下/从内到外）

↓

【配件层】附属道具/装饰物

↓

【底座层】底座/环境/支撑结构

↓

【收束层】质感封装 + 背景声明

↓

【输出层】三轨并行表达

```

| 装备道具 | 奇幻武器设计，道具概念图 |

| 产品设计 | 工业产品设计，产品渲染图 |

| 潮流公仔 | BE@RBRICK风格，KAWS风格，盲盒公仔设计 |

**本图定性**：

> Q版卡通设计，幻想形象，可爱的风格化乙烯基玩偶全身照，大头小身

## **六、拆解层：分区零件清单**

**核心技术：分区扫描**

按照固定顺序，逐区域描述每个零件。

### **分区扫描顺序**

**人物类**：

```

<TEXT>

头部整体 → 面部 → 头发 → 耳朵 → 头饰/角/帽

→ 颈部 → 上身服装 → 下身服装 → 手部 → 腿部/鞋靴

→ 飘带/披风/尾巴 → 手持道具

```

**物品类**：

```

<TEXT>

整体轮廓 → 顶部 → 主体结构 → 细节装饰 → 底座/支撑

```

### **零件描述公式**

```

<TEXT>

[部位名] + [形态描述] + [颜色] + [材质/质感] + [装饰细节]

```

**示例拆解（本图）**：

| **分区** | **零件描述** |

| --- | --- |

| **位置** | 坐落在一个风格化的、长满苔藓的树枝/圆木上 |

| **头发** | 光滑、蓬松的白色头发，带有柔和的卷曲 |

| **耳朵** | 长长的尖兔耳/鹿耳 |

| **头饰/角** | 巨大、复杂的鹿角，呈棕色和苔绿色渐变，点缀着精致的白色花朵和蓝色珠子挂件 |

| **上身** | 精致的冬日服装，颜色为淡蓝色和奶油白色，一件带有白色毛皮/绒毛边饰的多层高领外套 |

| **鞋靴** | 厚重的白色和蓝色靴子 |

| **飘带** | 长长、飘逸的淡蓝色/白色丝带从她的头部和衣服上垂下 |

## **九、收束层：质感封装 + 背景**

**用风格标签"封装"整体质感，声明背景处理。**

### **质感标签库**

| **类别** | **常用标签** |

| --- | --- |

| **整体风格** | 可爱的超现实主义艺术风格，梦幻、可爱、华丽 |

| **质感定性** | Q版手办质感，粘土人风格，收藏级摆件 |

| **渲染风格** | CG渲染感，3D渲染，辛烷渲染，虚幻引擎 |

| **细节强调** | 细节精致，立体感强，纤毫毕现 |

| **参考锚定** | 墨瞳MT，Good Smile，寿屋风格 |

### **背景声明**

| **需求** | **句式** |

| --- | --- |

| 纯色背景 | 背景是干净、极简的纯淡灰/米白色 |

| 渐变背景 | 背景是柔和的蓝紫色渐变 |

| 透明/可抠 | 透明背景，PNG格式 |

| 场景背景 | 背景是模糊的[场景描述] |

**本图收束**：

> 可爱的超现实主义艺术风格，整体造型梦幻、开心，可爱、华丽，呈现Q版手办质感，墨瞳MT，细节精致，立体感强，收藏级摆件，粘土人风格，CG渲染感。突出主体，背景是干净、极简的纯淡灰/米白色

## **十二、完整输出示例**

---

**【创作意图】**

```

<TEXT>

图像类型：Q版手办设计

主体：冬日星辰鹿仙子

风格定性：Q版卡通，乙烯基玩偶，大头小身

分区清单：头发、耳朵、鹿角、上身服装、靴子、飘带

配件：蓝色小花瓶、树枝、流苏挂件

底座：苔藓树枝/圆木、白色花朵

质感标签：墨瞳MT，粘土人风格，CG渲染感

背景：纯淡灰/米白

```

---

**【表达A：自然语言】**

> Q版卡通设计，幻想形象。一个可爱的、风格化的乙烯基玩偶全身照，大头小身比例，描绘的是一个冬日/星辰鹿仙子。她安静地坐着，眼睛闭合，坐落在一个风格化的、长满苔藓的树枝/圆木上。她有着光滑、蓬松的白色头发，带有柔和的卷曲，以及长长的尖兔耳/鹿耳。她的头部装饰着巨大、复杂的鹿角，呈棕色和苔绿色渐变，点缀着精致的白色花朵和蓝色珠子挂件。她穿着一套精致的冬日服装，颜色为淡蓝色和奶油白色，包括一件带有白色毛皮/绒毛边饰的多层高领外套，以及厚重的白色和蓝色靴子。她有长长、飘逸的淡蓝色/白色丝带从她的头部和衣服上垂下。树枝旁边放着一个小巧、透明的蓝色花瓶，花瓶里插着一根风格化的树枝和一个蓝色流苏挂件。底座包括雕刻的白色花朵和苔藓。可爱的超现实主义艺术风格，整体造型梦幻、开心、可爱、华丽，呈现Q版手办质感，墨瞳MT，细节精致，立体感强，收藏级摆件，粘土人风格，CG渲染感。突出主体，背景是干净、极简的纯淡灰/米白色。
