# Prompt Library Guide

> 如何管理和使用你的 Prompt 库

## Prompt 模板结构

```markdown
---
title: ""
version: 1.0
model: gpt-4o / claude-3.5 / ...
tags: [prompt/category]
---

# Prompt 名称

## 适用场景

## System Prompt

```

## 分类规则

- **Research**: 研究分析类 Prompt
- **Workflow**: 工作流优化类 Prompt
- **Automation**: 自动化任务类 Prompt
- **Business**: 商业分析/决策类 Prompt
- **Content**: 内容创作类 Prompt
- **Video Script**: 视频脚本类 Prompt
- **Cover Design**: 封面/视觉设计类 Prompt

## 版本管理

每次有效迭代后更新版本号：1.0 -> 1.1 -> 2.0
