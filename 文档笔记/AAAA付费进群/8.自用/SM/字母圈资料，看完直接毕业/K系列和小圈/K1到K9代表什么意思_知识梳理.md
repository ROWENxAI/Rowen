# K1到K9代表什么意思.docx 知识梳理

> 来源: `K1到K9代表什么意思.docx` | 字数: 449

## 内容全文

K1指的是侍奴；

K2指的是跪奴；

K3指的是刑奴，也叫sp奴；

K4指的是绳奴；

K5指的是猪奴；

K6指的是性奴；

K7指的是妻奴，如果是男的就是夫奴；

K8指的是猫奴；K9指的是狗奴。

除此之外，还有一个K0，指的是厕奴。

K3、k6、k9是比较常见的类型，k0偏重度的类型。

字母圈k0-k9的来源：

K0 -> 0像不像张大嘴的人？这是厕奴

K1 -> 1像不像一个站立服侍的人？这是侍奴

K2 -> 2像不像跪着的人？这是跪奴

K3 -> 3像不像一个屁股？这是sp奴，也是刑奴

K4 -> 4像不像被绑着的人呀？这是绳奴

K5 -> 5像不像大腹便便的一个胖子呀？这是猪奴

K6 -> 6就是six嘛，谐音就是sex，所以k6是Xing奴

K7 -> 7嘛，谐音妻，这是妻奴，如果是个男的就是夫奴

K8 -> 离k9最近，k9是狗奴，那k8就是猫奴啦!

K9 -> canine (英文谐音)， canine原意指犬科动物，在SM中则为人形犬，因此k9指狗奴
