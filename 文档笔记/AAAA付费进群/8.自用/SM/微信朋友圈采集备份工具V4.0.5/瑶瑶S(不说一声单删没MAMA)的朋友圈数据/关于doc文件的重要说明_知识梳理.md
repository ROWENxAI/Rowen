# 关于doc文件的重要说明.txt 知识梳理

> 来源: `关于doc文件的重要说明.txt` | 字数: 206

## 内容全文

1、本软件导出的doc文件必须使用wps软件才能正常显示，wps官方下载：https://platform.wps.cn/

2、使用wps打开当前软件导出的doc文件后，请点击wps界面右下角的【页面视图】图标，切换到标准word页面

3、当前doc文档中的图片为本地调用，建议打开doc文件后，另存为新的doc文档（文件类型必须选择doc格式，且选择新的保存位置），这样可以将图片一并保存到新的doc文档中
