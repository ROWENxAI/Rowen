## 第一章：出海 AIP 基础准备工作

### 1.1 心理建设
- 出海不是逃避，是降维打击
- 国内内容创作经验在海外有信息差优势
- 接受前期粉丝少的冷启动期

## 第二章：出海 AIP 的定位

### 2.1 自媒体不可能三角
- 泛流量、高变现、高信任不可能同时做到
- 泛流量不值钱：吸引来的不是目标受众
- 要走 IP 路线：建立信任 + 垂直变现

### 2.2 适合出海 AIP 的黄金赛道
- AI / 编程 / 工具类（推特主流人群）
- 商业 / 创业 / 营销
- 个人成长 / 效率方法

### 2.3 如何找对自己的方向
- 结合自身技能和经验
- 分析目标平台人群画像
- 找到需求和供给的缺口

## 第三章：核心阵地 X 平台的基础认知

### 3.1 中推 vs 英推
- 中文推特：圈子小但粘性高，适合华人项目变现
- 英文推特：流量大但竞争激烈，需英文能力

### 3.2 2025 X 平台算法规则
- 算法倾向于互动率高的内容
- 发布时间、频率、互动策略

### 3.3 内容运营红线
- 国内：敏感话题、政治、版权
- 国外：SPAM、抄袭、虚假信息

### 3.4 账号开业准备清单
- 头像 / Banner / Bio / Pin 帖 / 置顶链接
- 像开店一样运营账号

## 第四章：如何运营图文短内容

### 4.1 四种常见的图文短内容
1. 故事型 —— 涨粉效果最好
2. 列表型 —— 流量最高
3. 观点型 —— 建立 IP 调性
4. 教程型 —— 建立专业信任

### 4.2 如何写高涨粉的故事
- 开头钩子（前 3 秒决定是否继续读）
- 情节冲突（制造悬念和共鸣）
- 结尾反转或金句

### 4.3 如何写高流量列表
- 数字 + 利益点（如 "5 个让我月入过万的工具"）
- 每一条有实质内容而非凑数
- 格式清晰易读

### 4.4 如何发表观点或问题
- 观点型内容最容易引发讨论
- 提问式互动提高评论区活跃度
