# AGENTS.md

> AI Agent 操作本知识库时的特殊规则。

## 启动流程

当 AI Agent 在本知识库中工作时，请按此流程操作：

1. 读取 `98-AI-Context/AI Context Index.md` 获取上下文概述
2. 读取 `98-AI-Context/About Me.md` 了解用户
3. 读取 `98-AI-Context/Current Focus.md` 了解当前焦点
4. 读取 `98-AI-Context/AI Operating Context.md` 了解操作规则
5. 加载 `97-AI-Memory/` 中的相关记忆

## 编辑约定

- 所有笔记使用 UTF-8 编码
- Obsidian 内部链接使用 `[[wikilink]]` 格式
- 标签使用 `tag1/tag2` 层级格式
- 每个笔记至少一个 `type/xxx` 标签和一个 `topic/xxx` 标签

## 禁止行为

- 不删除原始研究资料
- 不擅自修改 AI Memory 中的历史记录（仅追加新条目）
- 不在 Research 目录下创建非研究类文件
