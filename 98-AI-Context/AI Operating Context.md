# AI Operating Context

> 为 AI Agent 提供的操作指南。

## 知识库结构速览

```
H:\转录├── 00-Inbox/          # 收集箱
├── 04-Research/       # 研究笔记
├── 05-Content/        # 内容创作
├── 06-Projects/       # 项目管理
├── 89-Prompts/        # Prompt 库
├── 90-Templates/      # 模板
├── 97-AI-Memory/      # AI 长期记忆
├── 98-AI-Context/     # AI 上下文
└── 99-Archive/        # 归档
```

## 工作原则

1. **长期记忆优先**：重要信息写入 97-AI-Memory，而非留在对话中
2. **上下文初始化**：新对话先读 98-AI-Context/
3. **保留原文**：研究笔记不删减原文
4. **去重**：入库前检查是否已有内容
5. **链接优先**：尽量使用内部链接连接相关笔记

## 操作权限

- 可创建、编辑 00-99 目录下的文件
- 不可删除其他目录的文件（除非明确要求）
- 清理 Inbox 时需确认
